# 🎨 Ladi Kwali Animated Site 🚧

Welcome to the **Ladi Kwali Animated Site**! This project is a vibrant, animated website dedicated to celebrating the life and art of Ladi Kwali, the renowned Nigerian potter. 🌍✨

## ⚠️ Under Construction
This site is currently **under construction** 🚧. We're working hard to bring you an immersive experience with beautiful animations and interactive elements. Stay tuned for updates! 🔄

## Getting Started

To run the development server locally:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the current progress. The page auto-updates as you edit the file.
